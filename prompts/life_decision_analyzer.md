# 人生决策分析维度

## 分析框架

### 1. 可逆性检查（贝佐斯 Type 1/Type 2）

**Type 2（可逆决策）— 可以大胆做：**
- 换工作（换回来也行）
- 搬到一个新城市
- 学一个新技能
- 开始一个副业
- 这些决策做了之后可以撤回，代价有限

**Type 1（不可逆决策）— 要非常谨慎：**
- 辞职创业（放弃稳定收入）
- 移民（切断根基）
- 结婚/离婚（法律和社会关系绑定）
- 卖房 all-in 某件事
- 这些决策一旦做了很难撤回

**可逆性评分**：
- 完全可逆 → +20 梭哈指数
- 部分可逆 → +10
- 几乎不可逆 → -10
- 完全不可逆 → -20

### 2. 最坏情况推演

用"恐惧设定"（Fear-Setting）方法：

1. **如果我梭哈了，最坏会怎样？**
   - 列出所有可能出错的事
2. **这些最坏情况发生了，我能做什么？**
   - 每一项有没有应对方案
3. **如果不梭哈，1 年后 / 5 年后我会怎样？**
   - 不行动的代价
4. **我在害怕什么？这个恐惧合理吗？**
   - 区分真实风险和想象中的恐惧

### 3. 机会成本分析

- 梭哈：投入 X 年 / X 万 / X 精力 → 可能的收益
- 不梭哈：保持现状 → 1 年后的状态 / 5 年后的状态
- 第三选择：有没有不用 all-in 但也能推进的中间路线？

### 4. 人生阶段适配

| 阶段 | 梭哈容忍度 | 原因 |
|------|-----------|------|
| 20-25 岁 | 高 | 试错成本低，时间充裕 |
| 25-35 岁 | 中高 | 有一些积累，但还有时间恢复 |
| 35-45 岁 | 中 | 家庭责任增加，需平衡 |
| 45 岁以上 | 低偏中 | 时间和健康是稀缺资源 |

这不是绝对的——只是说不同阶段的容错率不同。

### 5. 情感 vs 理性判断

**理性决策信号**（加分）：
- "我考虑了 X 个月"
- "我咨询了 X 个人"
- "我做了预算/计划"
- "最坏情况我能接受"

**情感决策信号**（减分）：
- "我受够了" / "不想再忍了"
- "再不做就来不及了"
- "我不知道，就是想做"
- 重大人生变故后的冲动（失恋后辞职、被裁后移民）

### 6. 执行路径建议

如果建议梭哈：
- 分阶段还是一次性？建议分阶段
- 第一步具体做什么？
- 什么信号表示该停止？
- 什么信号表示该加速？

如果不建议梭哈：
- 有没有折中方案？
- 需要满足什么条件才值得做？
- 建议观察多长时间？
