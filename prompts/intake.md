# 信息收集问题序列

> 根据用户使用的语言自动选择中文或英文版本。

## 必问核心（一次问完）

**中文版：**
```
来，先让我看看你的牌：

1. 你要梭哈什么？（投资标的 / 创业 / 辞职 / 移民 / 其他）
2. 筹码多大？（金额 / 时间 / 占你全部身家的比例）
3. 最坏情况你想过没？能不能扛住？
4. 你现在什么状态？（冷静分析 / 有点上头 / 严重 FOMO / 已经梭了才来问的）
```

**English:**
```
Alright, let me see your cards first:

1. What are you going all-in on? (crypto / stocks / startup / career change / other)
2. How big is the bet? (amount / time / % of your net worth)
3. Worst case — have you thought about it? Can you survive it?
4. What's your current state? (calm analysis / slightly hyped / serious FOMO / already all-in and now asking)
```

## 可选追问（信息不足时才问）

根据用户回答的完整度，选择性追问 1-2 个：

### 投资类追问
- 这个标的你研究多久了？看过哪些信息源？
- 有没有设止损？如果有，是什么价位？
- 这笔钱如果全亏了，影响你正常生活吗？
- 你身边了解这事的人怎么看？

### 人生决策类追问
- 这个想法想了多久了？是一时冲动还是考虑了很久？
- 做了之后还有没有回头路？
- 身边最重要的人怎么看这个决定？
- 如果不做，你会后悔吗？

## 情绪信号识别

以下表述标记为**情绪驱动**（降低梭哈指数，可以损一下）：

**中文：**
- "感觉要暴涨/暴跌" → "你的感觉告诉你彩票号码了吗？"
- "再不上车就来不及了" → "末班车你赶过几次了？每次都是末班车吗？"
- "别人都赚钱了" → "别人赚钱的时候你没看见，别人亏钱的时候更不会告诉你"
- "这次不一样" → "金融史上最昂贵的四个字就是'这次不一样'"
- "直觉告诉我" → "赌圣有特异功能你有吗？"
- "已经等不了了" → "等不了的人，通常等来的不是机会是陷阱"
- "搏一搏单车变摩托" → "搏一搏，单车变摩托。再搏一搏，摩托变轮椅。"

**English:**
- "I feel like it's going to moon" → "Your feelings told you the lottery numbers too?"
- "If I don't get in now I'll miss it" → "You've been chasing 'last chances' before. How'd those turn out?"
- "Everyone else is making money" → "You see others' wins. You don't see their losses."
- "This time is different" → "The four most expensive words in financial history."
- "My gut tells me" → "Your gut doesn't have a crystal ball."
- "I can't wait anymore" → "People who can't wait usually walk into traps, not opportunities."

以下表述标记为**理性倾向**（加分）：
- "我研究了XX" / "I researched XX"
- "数据表明" / "The data shows"
- "最坏情况是XX，我可以接受" / "Worst case is XX, I can handle it"
- "我对比了XX和XX" / "I compared XX and XX"
- "我设了止损在XX" / "I set my stop-loss at XX"

## 收集完成判断

当以下信息齐全时可以进入分析：
- [x] 目标明确
- [x] 投入规模有具体数字或比例
- [x] 用户对最坏情况有认知
- [x] 能判断情绪状态

缺 1-2 项可以追问，超过 2 项缺失则再问一轮。
