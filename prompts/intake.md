# 信息收集问题序列

## 必问核心（一次问完）

```
来，聊清楚再帮你分析：

1. 你要梭哈什么？（投资标的 / 创业 / 辞职 / 移民 / 其他）
2. 投入规模多大？（金额 / 时间 / 占你全部资源的比例）
3. 最坏结果是什么？你能承受吗？
4. 现在什么状态？（FOMO / 理性分析 / 被形势推着走 / 兴奋上头）
```

## 可选追问（信息不足时才问）

根据用户回答的完整度，选择性追问 1-2 个：

### 投资类追问
- 这个标的你研究多久了？看过哪些信息源？
- 有没有设止损？如果有，是什么价位？
- 这笔钱如果全亏了，影响你正常生活吗？
- 你身边了解这事的人怎么看？

### 人生决策类追问
- 这个想法想了多久了？是一时冲动还是考虑了很久？
- 做了之后还有没有回头路？
- 身边最重要的人怎么看这个决定？
- 如果不做，你会后悔吗？

## 情绪信号识别

以下表述标记为**情绪驱动**（降低梭哈指数）：
- "感觉要暴涨/暴跌"
- "再不上车就来不及了"
- "别人都赚钱了"
- "这次不一样"
- "直觉告诉我"
- "已经等不了了"
- "搏一搏单车变摩托"

以下表述标记为**理性倾向**（加分）：
- "我研究了XX"
- "数据表明"
- "最坏情况是XX，我可以接受"
- "我对比了XX和XX"
- "我设了止损在XX"

## 收集完成判断

当以下信息齐全时可以进入分析：
- [x] 梭哈目标明确
- [x] 投入规模有具体数字或比例
- [x] 用户对最坏情况有认知
- [x] 能判断情绪状态

缺 1-2 项可以追问，超过 2 项缺失则再问一轮。
