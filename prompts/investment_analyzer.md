# 投资决策分析维度

## 分析框架

### 1. 预期收益分析

三种情景分析：

| 情景 | 概率 | 预期收益 | 触发条件 |
|------|------|---------|---------|
| 乐观 | 估计% | 估计收益 | 什么情况下会发生 |
| 中性 | 估计% | 估计收益 | 最可能的情况 |
| 悲观 | 估计% | 估计收益 | 什么情况下会发生 |

**期望值 = 概率1 × 收益1 + 概率2 × 收益2 + 概率3 × 收益3**

如果期望值为正，说明从概率上值得做；如果为负，说明从概率上不值得。

### 2. 下行风险分析

- **最大亏损**：最坏情况亏多少？（本金归零？负债？）
- **亏损影响**：对生活的影响程度（不影响 > 痛但能接受 > 致命打击）
- **可逆性**：亏了之后能不能恢复？需要多久？
- **连带风险**：会不会影响家庭、信用、其他投资？

### 3. 凯利公式评估

简化版仓位建议：

```
f* = (bp - q) / b

其中：
- f* = 建议投入比例
- b = 赔率（盈利/亏损比）
- p = 胜率
- q = 1 - p（败率）
```

实际建议：凯利值的一半（半凯利）作为安全边际。

### 4. 机会成本

- 同样的钱和时间，如果不梭哈这个，最优替代选择是什么？
- 替代选择的收益/风险比如何？
- 梭哈的机会成本是否可接受？

### 5. 市场环境评估

- 当前是牛市/熊市/震荡期？
- 大环境对该标的是有利还是不利？
- 有没有即将到来的黑天鹅事件（政策、利率、地缘）？

### 6. 仓位建议逻辑

| 梭哈指数 | 仓位建议 |
|----------|---------|
| 80-100 | 可以重仓（50-80%可用资金） |
| 60-79 | 中等仓位（20-50%可用资金） |
| 40-59 | 轻仓试探（5-20%可用资金） |
| 0-39 | 观望（0%，不入场） |

**硬性规则**：
- 永远不建议 100% 全仓单一标的
- 如果是杠杆/借贷梭哈，额外扣 20 分梭哈指数
- 如果影响基本生活（房租、房贷、医疗），直接不建议

### 7. 止损建议

如果建议入场，必须给出：
- 建议止损位（具体价格或百分比）
- 建议止盈策略（分批止盈/目标价）
- 最大持有时间建议
