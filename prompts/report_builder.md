# 最终报告生成模板

## 报告结构

```
━━━ 梭哈分析报告 ━━━

📊 梭哈指数：XX/100
{一句话定性：可以梭哈 / 可以试，控制仓位 / 别梭，小仓位观察 / 别梭}

├─ 分析过程
│  ├─ 预期收益：
│  │  乐观（xx%概率）：收益 xxx
│  │  中性（xx%概率）：收益 xxx
│  │  悲观（xx%概率）：亏损 xxx
│  │  期望值：xxx
│  │
│  ├─ 下行风险：
│  │  最坏情况：xxx
│  │  可逆性：{完全可逆 / 部分可逆 / 不可逆}
│  │  致命程度：{不影响 / 痛但能接受 / 致命}
│  │
│  ├─ 概率评估：
│  │  基于已知信息的胜率判断：xx%
│  │  判断依据：{一句话}
│  │
│  ├─ 机会成本：
│  │  不梭哈的最优替代：xxx
│  │  机会成本可接受吗：{是/否，因为...}
│  │
│  └─ 情绪状态：
│     判断：{理性 / 轻度情绪化 / 明显 FOMO / 冲动}
│     {如果非理性：建议先冷静 xx 小时再做决定}
│
├─ 多视角碰撞
│  ├─ 🟢 乐观派说：{2-3句话核心观点}
│  ├─ 🔴 悲观派说：{2-3句话核心观点}
│  └─ 🟡 现实派说：{2-3句话综合判断}
│
├─ ✅ 核心支持理由
│  1. {最有力的一条}
│  2. {第二有力的一条}
│  3. {第三有力的一条}
│
├─ ❌ 核心反对理由
│  1. {最危险的一条}
│  2. {第二危险的一条}
│  3. {第三危险的一条}
│
├─ ⚠️ 关键风险点
│  {用户最可能忽略的 1-2 个风险}
│
├─ 📝 如果梭哈，建议策略：
│  投入比例：xx% 可用资金

│  止盈策略：{分批/目标价}
│  分批还是一次性：{建议}
│
└─ 💡 降低风险的 3 个方法
   1. {最实用的方法}
   2. {第二实用的方法}
   3. {第三实用的方法}

━━━━━━━━━━━━━━━━━
🎯 一句话结论：
{直白犀利，不废话。比如：
- "别梭，你在 FOMO"

- "先别急，等你冷静下来再看这份报告"}
━━━━━━━━━━━━━━━━━
```

## 梭哈指数计算规则

基础分 50 分，根据以下维度加减：

### 加分项（每项 +5 到 +15）
- 不对称收益（上行远大于下行）：+10~15
- 概率明显有利：+10
- 决策可逆：+10
- 有充分研究：+5~10
- 在能力圈内：+5
- 理性决策状态：+5
- 有明确止损计划：+5

### 减分项（每项 -5 到 -20）
- 下行可能致命：-20
- 使用杠杆/借贷：-20
- 情绪驱动/FOMO：-10~15
- 信息不充分：-10
- 不可逆决策：-10
- 无止损计划：-5
- 影响基本生活：-15

最终分数 clamp 到 0-100。

## 报告风格要求

1. 分析过程要**有逻辑、有数据**（即使是估算），不能全是定性描述
2. 多视角碰撞要**有交锋**，不是三个平行独立的观点
3. 一句话结论必须**直白犀利**，不骑墙
4. 如果是高风险决策，关键警示要醒目
5. 全篇用语平实，不用"综上所述"、"值得一提的是"这种废话
