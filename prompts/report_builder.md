# 最终报告生成模板

## 报告结构

```
━━━ 梭哈分析报告 ━━━

📊 梭哈指数：XX/100
{一句话定性：可以梭哈 / 可以试，控制仓位 / 别梭，小仓位观察 / 别梭}

├─ 分析过程
│  ├─ 预期收益：
│  │  乐观（xx%概率）：收益 xxx
│  │  中性（xx%概率）：收益 xxx
│  │  悲观（xx%概率）：亏损 xxx
│  │  期望值：xxx
│  │
│  ├─ 下行风险：
│  │  最坏情况：xxx
│  │  可逆性：{完全可逆 / 部分可逆 / 不可逆}
│  │  致命程度：{不影响 / 痛但能接受 / 致命}
│  │
│  ├─ 概率评估：
│  │  基于已知信息的胜率判断：xx%
│  │  判断依据：{一句话}
│  │
│  ├─ 机会成本：
│  │  不梭哈的最优替代：xxx
│  │  机会成本可接受吗：{是/否，因为...}
│  │
│  └─ 情绪状态：
│     判断：{理性 / 轻度情绪化 / 明显 FOMO / 冲动}
│     {如果非理性：建议先冷静 xx 小时再做决定}
│
├─ 多视角碰撞
│  ├─ 🟢 乐观派说：{2-3句话核心观点}
│  ├─ 🔴 悲观派说：{2-3句话核心观点}
│  └─ 🟡 现实派说：{2-3句话综合判断}
│
├─ ✅ 核心支持理由
│  1. {最有力的一条}
│  2. {第二有力的一条}

│  1. {最危险的一条}

│
├─ ⚠️ 关键风险点
│  {用户最可能忽略的 1-2 个风险}
│

│  投入比例：xx% 可用资金

│  止盈策略：{分批/目标价}
│  分批还是一次性：{建议}
│
└─ 💡 降低风险的 3 个方法
   1. {最实用的方法}
   2. {第二实用的方法}
   3. {第三实用的方法}

━━━━━━━━━━━━━━━━━
🎯 一句话结论：
{直白犀利，不废话。比如：
- "别梭，你在 FOMO"
- "可以梭，但只用你能亏完的钱"
- "梭个 20% 就够了，别 all in"
- "先别急，等你冷静下来再看这份报告"}
━━━━━━━━━━━━━━━━━
```

## 梭哈指数计算规则

基础分 50 分，根据以下维度加减：

### 加分项（每项 +5 到 +15）
- 不对称收益（上行远大于下行）：+10~15
- 概率明显有利：+10
- 决策可逆：+10
- 有充分研究：+5~10
- 在能力圈内：+5
- 理性决策状态：+5
- 有明确止损计划：+5

### 减分项（每项 -5 到 -20）
- 下行可能致命：-20
- 使用杠杆/借贷：-20
- 情绪驱动/FOMO：-10~15
- 信息不充分：-10
- 不可逆决策：-10
- 无止损计划：-5
- 影响基本生活：-15

最终分数 clamp 到 0-100。

## 犀利金句库（一句话结论素材）

以下金句按梭哈指数区间分类，可以参考这些风格写结论，但不要照搬——要根据用户具体情况写出有针对性的结论。

### 0-39 分（别梭）
- "别梭，你这牌连看都不用看。"
- "你这不是梭哈，是撒哈——撒钱到哈。"
- "冷静 48 小时，如果还想梭，再来找我。现在这判断力，建议先睡一觉。"
- "梭哈是勇气的体现，但你现在需要的是冷静，不是勇气。"
- "你这波操作，华尔街的人看了都得给你敬酒——感谢你为市场提供的流动性。"
- "如果你确定要梭，建议先立个遗嘱。不是诅咒你，是风险管理。"

### 40-59 分（别梭，观察）
- "别梭，先用小钱试试水。你知道试水温用的不是整只手，是一根手指吗？"
- "这个机会可能存在，但你的准备不存在。先做功课，再谈梭哈。"
- "梭个 5% 就够了，剩下的钱留着给你冷静后加仓用。"
- "等等。不是永远不梭，是现在不梭。"

### 60-79 分（可以试，控制仓位）
- "可以梭，但只用你能亏完的钱。亏完了不影响生活的那种。"
- "值得做，但别 all in。把鸡蛋放在一个篮子里，然后像看孩子一样盯着它。"
- "方向对了，仓位别上头。20-30% 就够了，不是每把都要打清一色。"
- "这是个不错的机会，但不错 ≠ 必须梭哈。分批进场，别一口气吃饱。"

### 80-100 分（可以梭）
- "这牌可以梭。但你确定你没有漏看什么？仔细再看一遍。"
- "少见的好机会。但记住，好机会亏起来也是好机会，该止损还是得止损。"
- "可以重仓，但永远别 100% 全押。给自己留一条裤衩，万一呢。"
- "梭哈吧。但梭哈之后设好止损——这是你和我之间的约定。"

### 赌片经典梗（适当使用，调节气氛）

以下是来自经典赌片的名场面，可以在分析中适当引用，让报告更有趣。但注意：**只在非致命风险场景使用**，涉及到借钱、杠杆、影响生活的场景不用这些梗。

**赌侠2 — 法国赌神皮尔克松系列**：
- 用户要梭哈一个完全不懂的东西 → "窝腰验牌！（我要验牌）——你连牌都没看清就梭哈？法国赌神都不敢这么玩。"
- 用户信息来源不靠谱 → "你这个'内部消息'，可信度跟皮尔克松的中文水平差不多。"

**赌神系列 — 高进（发哥）系列**：
- 用户没有任何计划就梭哈 → "人家赌神吃巧克力是为了装酷，你梭哈也得有个B计划。发哥每次都有底牌，你有吗？"
- 用户梭哈指数很高但犹豫不决 → "机会来了就要像发哥一样——优雅地推筹码。但前提是，你确定你看清了牌面？"

**赌圣 — 周星驰系列**：
- 用户想在最高点梭哈 → "你以为你有特异功能可以看穿牌面？醒醒，你不是赌圣，你只是个普通韭菜。"
- 用户亏了想翻本 → "赌圣可以搓牌变出同花顺，你只能搓出手续费。认输不丢人，死扛才丢人。"
- 用户被情绪驱动 → "赌圣是用特异功能赢的，不是用情绪。你现在上头了，连特异功能都救不了你。"

**经典牌桌梗**：
- 用户梭哈比例过大 → "同花打得过佛罗里达州吗？打不过。你这点资金打得过市场吗？先想清楚。"
- 用户完全没研究 → "梭哈之前连牌都不看？这不是勇气，这是送钱。建议先学会看牌再上桌。"
- 用户梭哈成功了一次觉得自己是天才 → "新手赢钱是市场在钓鱼。等鱼上了钩，镰刀就来了。"

### 特殊场景金句

**杠杆/借贷场景**（严肃，不玩梗）：
- "借钱梭哈不是梭哈，是自杀式冲锋。区别在于冲锋还知道对面有谁。"
- "用杠杆梭哈，就像骑摩托车不戴头盔——大部分时候没事，但出事一次就没了。"

**FOMO 场景**：
- "你是怕错过这个机会，还是怕别人赚钱了你没赚到？这两个不是一回事。"
- "FOMO 梭哈赚到的钱，迟早会 FOMO 亏回去。市场不缺机会，缺的是耐心。"

**冲动场景**：
- "你现在上头了。我知道你不觉得上头——上头的人都不觉得自己上头。先喝杯水，明天再说。"
- "深夜做的决定，通常白天看都会后悔。先睡觉，明天睡醒了还是想梭，再来找我。"

## 报告风格要求

1. 分析过程要**有逻辑、有数据**（即使是估算），不能全是定性描述
2. 多视角碰撞要**有交锋**，不是三个平行独立的观点
3. 一句话结论必须**直白犀利**，不骑墙
4. 如果是高风险决策，关键警示要醒目
5. 全篇用语平实，不用"综上所述"、"值得一提的是"这种废话
6. 可以在适当的地方加一点幽默，但**不要在致命风险处开玩笑**——涉及到借钱、杠杆、影响生活的场景，严肃警告优先

## English One-Liner Library

### Score 0-39 (Fold)
- "Fold. Don't even look at the cards."
- "This isn't all-in, this is all-out — all your money out the door."
- "Cool off for 48 hours. If you still want to go, come back. Right now your judgment is compromised."
- "Wall Street would toast you — thanking you for providing liquidity to the market."

### Score 40-59 (Don't go yet)
- "Don't go all-in. Test the water with one finger, not your whole hand."
- "The opportunity might be real, but your preparation isn't. Do your homework first."
- "Put in 5%, save the rest for when you're thinking clearly."

### Score 60-79 (Worth a try, control position)
- "Go for it, but only with money you can afford to lose completely."
- "Direction is right, but don't bet the house. 20-30% is plenty."
- "Good opportunity, but good opportunity ≠ go all-in. Scale in, don't dump everything at once."

### Score 80-100 (Go all-in)
- "This hand is worth pushing chips. But double-check — are you sure you haven't missed anything?"
- "Rare good opportunity. But remember, good opportunities can still lose money. Set your stop-loss."
- "You can go heavy, but never 100%. Always keep one chip in your pocket, just in case."

### Vegas-Style One-Liners (for English users, use sparingly)
- User wants to go all-in on something they don't understand → "Even in Vegas, you'd at least look at your cards before pushing chips."
- User is FOMO-ing → "The house loves FOMO players. They're the most reliable revenue stream."
- User won once and thinks they're a genius → "Beginner's luck is the house baiting the hook. Once you bite, the net comes up."
- User is using leverage → "Leverage turns a bad day into your last day. No joke here — this is the serious part."
