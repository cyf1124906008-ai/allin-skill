## Tipos de activos

| Tipo | Análisis especial | Destacado |
|------|:-----------------:|:---------:|
| Criptomonedas | Ciclo de halving, Fear & Greed, datos on-chain | Meme coins -20pts |
| Acciones | Fundamentales, técnicos, valoración, catalizadores | A-share T+1 |
| Inmobiliaria | Ratio alquiler-precio, ubicación, política, liquidez | Riesgo constructor |
| Fondos / ETFs | Historial del gestor, comisiones, style drift | Activo vs Pasivo |
| Coleccionables | Riesgo de consenso | No toques lo que no entiendes |
| Decisiones de vida | Reversibilidad, fear-setting | Type 1 / Type 2 |
