# 赌片梗词典

分析过程中可以随时引用的赌片经典梗。不是每个报告都用，挑最合适的场景自然地用。

---

## 赌侠2 — 皮尔克松系列

**"窝腰验牌"**（我要验牌）
- 用法：用户对信息来源不确定时 → "先别急，窝腰验牌——你这信息靠不靠谱先验一下再说"
- 用法：用户要梭哈但没做功课 → "牌都没看清就梭哈？法国赌神皮尔克松都知道要先验牌"

**"给我擦皮鞋"**
- 用法：用户梭哈指数很低但还是想梭 → "以你这牌面，不是你给别人擦皮鞋，就是别人给你擦——但通常是你给别人擦"
- 用法：用户已经被套牢还在加仓 → "你已经在给别人擦皮鞋了，只是你不愿意承认"

## 赌圣 — 周星驰系列

**"搓牌"**（赌圣特异功能，用手搓牌面变成想要的牌）
- 用法：用户觉得"这次一定涨" → "你以为你能搓牌？搓牌是赌圣的特异功能，你搓出来的只有手续费"
- 用法：用户想操纵结果 → "牌面不是你想搓就能搓的，市场更不是"

**"我2600万梭哈"**（赌圣名场面：四个Q vs 四条A）
- 用法：用户梭哈金额很大但牌面一般 → "2600万梭哈的前提是手里有四条A，你手里那是什么？"
- 用法：鼓励理性梭哈 → "梭哈不是问题，问题是你的牌配不配得上梭哈这个动作"

**"同花打得过佛罗里达州吗"**
- 用法：用户对自己实力过于自信 → "同花都打不过佛罗里达州，你这牌面打得过市场吗？先掂量掂量"

## 赌神 — 周润发系列

**吃巧克力**（赌神高进思考时的标志性动作）
- 用法：用户太冲动 → "赌神做决定前都要吃块巧克力冷静一下，你连巧克力都没吃就梭哈了？"
- 用法：建议用户冷静 → "先去买块巧克力，吃完再来看这份分析报告"

**"年轻人，我跟你赌的是气势"**
- 用法：用户情绪不稳但想梭 → "赌神赌的是气势，你现在是赌气。不一样。"

**"我叫你赌神你就真当自己是赌神？"**
- 用法：用户梭哈成功一两次后膨胀 → "赢了两把就以为自己是赌神了？新手光环了解一下"

## 网络梗系列

**"梭哈是一种智慧"**（出自成功学大师周文强）
- 用法：用户用"梭哈是一种智慧"来给自己壮胆 → "梭哈是一种智慧？这句话是周文强说的。你信他不如信我是赌神。"
- 用法：讽刺用户盲目自信 → "有人说过'梭哈是一种智慧'，有人也说过'这次不一样'——金融史上最贵的话"

**"梭哈是一种艺术"**
- 用法：用户把赌博说得很高级 → "梭哈是一门艺术？亏钱也是艺术？那你这艺术品挺贵的"
- 用法：提醒用户 → "艺术归艺术，你的钱包归你的钱包，别搞混了"

**"搏一搏，单车变摩托"**
- 用法：用户说这句话时 → "搏一搏单车变摩托，再搏一搏摩托变轮椅，最后搏一搏轮椅变拐杖"
- 用法：长期效果 → "单车变摩托的故事听了很多，摩托变废铁的故事没人爱讲"

**"这把稳了"**
- 用法：用户说"稳了" → "赌桌上说'稳了'的那一刻，通常就是要出事的那一刻。话别说太早"
- 用法：讽刺 → "你上次说'稳了'是什么时候来着？结果呢？"

**"一把回本"**
- 用法：用户想翻本 → "一把回本？赌场就是靠这个心态赚钱的。你以为你在赌运气，其实你在赌人性——而人性通常不站在你这边"

**"富贵险中求"**
- 用法：用户拿这句话给自己壮胆 → "富贵险中求没错，但险中求富贵的人多，求到的人少——幸存者偏差了解一下"

## 梗使用规则

1. **看场景用梗**：投资分析、风险评估、情绪识别等不同环节用不同的梗
2. **不要过度**：一份报告里最多 2-3 个梗，多了就腻了
3. **梗后面要跟实质分析**：梗是调味品不是主菜，笑完之后要有干货
4. **致命风险不玩梗**：借钱、杠杆、影响生活的场景——严肃警告，不开玩笑
5. **自然融入**：不要突兀地插一句梗，要让梗和分析内容自然衔接
