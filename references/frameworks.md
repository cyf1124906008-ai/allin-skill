# 决策框架集合

分析过程中可引用的决策框架。不要生搬硬套，选择最适合当前场景的 2-3 个框架使用。

---

## 1. 凯利公式（Kelly Criterion）

**用途**：投资仓位管理

**公式**：f* = (bp - q) / b
- f* = 最优投入比例
- b = 净赔率（盈利金额 / 亏损金额）
- p = 胜率
- q = 1 - p

**实际应用**：
- 永远用"半凯利"（计算结果的一半）作为安全边际
- 如果算出来凯利值 < 0，说明从数学上就不该做

**局限**：假设你知道胜率和赔率——但现实中往往是估计值。

---

## 2. 期望值分析（Expected Value）

**用途**：量化决策的数学期望

**公式**：EV = P1×V1 + P2×V2 + ... + Pn×Vn
- Pi = 第 i 种结果的概率
- Vi = 第 i 种结果的价值

**判断标准**：
- EV > 0 → 从概率上值得做
- EV < 0 → 从概率上不值得做
- EV 接近 0 → 取决于你能不能承受波动

---

## 3. 贝佐斯 Type 1/Type 2 决策

**用途**：判断决策的可逆性

**Type 1（不可逆）**：
- 像走过一扇单向门
- 需要深思熟虑，充分收集信息
- 考虑最坏情况后再决定

**Type 2（可逆）**：
- 像走过一扇双向门
- 可以快速做决定
- 做错了可以退回来

**核心洞察**：大部分决策其实是 Type 2，但我们把它当成 Type 1 来纠结。

---

## 4. 杠铃策略（Barbell Strategy）

**来源**：塔勒布

**核心**：不走中间路线，而是两端配置：
- 90% 安全资产（确保不归零）
- 10% 高风险高收益（博不对称收益）

**应用到人生决策**：
- 保留一份稳定收入 + 用业余时间梭哈新方向
- 不要"半吊子"梭哈——要么保守，要么激进，不要不上不下

---

## 5. 达利欧的可信度加权

**来源**：瑞·达利欧《原则》

**核心**：不是一人一票，而是根据每个人的可信度加权：
- 在该领域有成功经验的人，权重更高
- 有逻辑和数据支撑的观点，权重更高

**应用**：当用户提到"别人说"或"网上说"时，评估信息源的可信度。

---

## 6. 反过来想（Inversion）

**来源**：查理·芒格

**核心**：不要想"怎么成功"，而是想"怎么避免失败"：
- "如果注定失败，会是因为什么？"
- "怎么才能确保亏钱？"→ 然后避免这些行为

**应用**：每个分析都应该包含"反向思考"环节。

---

## 7. 沉没成本忽略

**核心**：已经投入的不可回收的成本不应影响未来决策。

**常见陷阱**：
- "我已经投了 10 万了，现在退出太亏了" → 典型的沉没成本谬误
- "我已经花了一年准备，不能放弃" → 过去的一年花就花了，重要的是未来

---

## 8. 恐惧设定（Fear-Setting）

**来源**：蒂姆·费里斯

**三步法**：
1. 定义噩梦：最坏会怎样？（比你想的没那么可怕）
2. 预防措施：能做什么降低最坏情况发生的概率？
3. 修复方案：最坏情况发生了，能做什么补救？

**关键洞察**：不行动的代价往往比行动失败的代价更大，但我们高估了行动的风险，低估了不行动的风险。
