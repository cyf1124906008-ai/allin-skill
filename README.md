## 🧩 功能特性

### 🃏 赌神分析流程

每个梭哈决策经过 5 步流程：

| 步骤 | 说明 |
|------|------|
| 👁️ **看牌** | 收集4个核心信息：目标、筹码、最坏情况、情绪状态 |
| 🧠 **识牌** | 自动识别类型：投资 / 人生决策 / 混合 |
| 🧮 **算牌** | 5维度分析：预期收益、下行风险、概率、机会成本、情绪体检 |
| 🎭 **亮牌** | 3视角碰撞：乐观派 / 悲观派 / 现实派 |
| 🎰 **摊牌** | 梭哈指数 + 策略建议 + 赌神一句话 |

### 🏷️ 投资类型专属分析

| 类型 | 🔬 特殊维度 |
|------|---------|
| 🪙 **加密货币** | 减半周期、恐贪指数、链上数据、交易所风险、Meme币识别 |
| 📈 **股票** | PE/PB历史分位、ROE趋势、催化剂、A股涨跌停/T+1提醒 |
| 🏠 **房产** | 租售比、月供收入比、流动性陷阱、开发商风险 |
| 📊 **基金** | 基金经理3/5/10年业绩、最大回撤、费率对比 |
| 💎 **收藏品** | 共识价值 vs 真实价值、流动性极低警告 |

### 🎭 三位赌桌顾问

| 顾问 | 🧠 思维模型 | 🎤 风格 |
|------|---------|--------|
| 🟢 **乐观派** | 彼得·蒂尔（从0到1） | 不对称机会在哪里？🚀 |
| 🔴 **悲观派** | 塔勒布（黑天鹅） | 黑天鹅在哪里？尾部风险多大？🦢 |
| 🟡 **现实派** | 芒格（反过来想） | 如果注定失败，会输在哪？🤔 |

### 📊 梭哈指数评分

| 分数 | 🎲 判断 | 🎤 赌神的话 |
|------|------|---------|
| 🟢 80-100 | 可以梭哈 | 牌面好，筹码推出去 💪 |
| 🟡 60-79 | 可以试，控制筹码 | 值得跟，但别全押 ✋ |
| 🟠 40-59 | 别梭，先看看 | 先用小筹码探探路 🔍 |
| 🔴 0-39 | 别梭，收手吧 | 这牌连看都不用看，直接弃 🚫 |

### 📜 决策记录 + 复盘

- 📁 每次分析自动保存到本地
- 🔄 `/梭哈复盘` 查看历史记录和赌神命中率
- ✏️ 支持回填实际结果，验证分析准确率

## ⚠️ 注意事项

- 🎯 **赌神只是帮你算牌，最终决定权在你**：分析是工具，决策是你的人生
- 🛡️ **安全底线不可突破**：借钱梭哈、杠杆梭哈、拿保命钱梭哈——赌神也得拦你
- 📊 **分析质量取决于你给的信息**：你提供的信息越具体，牌面看得越清楚
- 🎲 **赌神也有看不准的时候**：命中率高不代表每次都对，最终决策后果自负
- 📋 **这不是投资建议**：这是一个帮你想清楚的工具，不是理财顾问
